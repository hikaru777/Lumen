# Lumen

**为沉浸音乐而生的菜单栏歌词应用。**

一款 macOS 应用：在菜单栏显示正在播放歌曲的歌词，并以随节拍律动的极光，让你不只是「听」，而是「沉浸其中」。

[![Download](https://img.shields.io/badge/Download-Lumen%201.0.1-blueviolet?style=for-the-badge)](https://github.com/hikaru777/Lumen/releases/latest)

</div>

---

## 功能特点

- **歌词显示** — 自动获取并同步显示 Apple Music / Spotify 正在播放歌曲的歌词
- **随节拍律动的极光** — 屏幕边缘的极光会对歌曲的鼓点、起音和人声做出反应
- **手动同步** — 没有歌词的歌曲，也可以粘贴歌词并打拍子，自行制作同步
- **完全本地运行** — 无内购，无需账号
- **支持 4 种语言** — 日本語 / English / 简体中文 / 한국어（自动跟随系统语言）

## 系统要求

- macOS 26.0 或更高版本

## 安装方法

1. 从[最新版本](https://github.com/hikaru777/Lumen/releases/latest)下载 `Lumen-x.x.x.dmg`
2. 打开 DMG，将 `Lumen.app` 拖入 `Applications`（应用程序）
3. 首次启动时，允许读取系统音频以及控制 Spotify / 音乐

> 已公证（Apple Developer ID），可直接通过 Gatekeeper。
