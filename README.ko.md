# Lumen

**음악에 빠져들기 위한, 메뉴 막대 가사 앱.**

재생 중인 곡의 가사를 메뉴 막대에 표시하고, 비트에 반응하는 오로라로 그저 듣는 것을 넘어 「빠져드는」 경험을 만드는 macOS 앱.

[![Download](https://img.shields.io/badge/Download-Lumen%201.0.1-blueviolet?style=for-the-badge)](https://github.com/hikaru777/Lumen/releases/latest)

</div>

---

## 특징

- **가사 표시** — Apple Music / Spotify에서 재생 중인 곡의 가사를 자동으로 가져와 동기화 표시
- **비트에 반응하는 오로라** — 곡의 킥, 온셋, 보컬에 반응하는 화면 가장자리의 오로라
- **수동 동기화** — 가사가 없는 곡도 붙여넣기 + 템포 입력으로 직접 동기화를 만들 수 있음
- **완전 로컬** — 결제 없음, 계정 불필요
- **4개 언어 지원** — 日本語 / English / 简体中文 / 한국어 (시스템 언어에 자동으로 따라감)

## 시스템 요구 사항

- macOS 26.0 이상

## 설치 방법

1. [최신 릴리스](https://github.com/hikaru777/Lumen/releases/latest)에서 `Lumen-x.x.x.dmg` 다운로드
2. DMG를 열고 `Lumen.app`을 `Applications`(응용 프로그램)으로 드래그
3. 첫 실행 시, 시스템 오디오 읽기와 Spotify / 음악 제어를 허용

> 공증 완료(Apple Developer ID). Gatekeeper를 그대로 통과합니다.
